1. Install Node.js
2. Run Node.js command prompt
3. cd to the root folder
4. npm init
5. npm install gulp -g
5. npm install gulp --save-dev
6. npm install gulp-babel --save-dev
7. npm install babel-preset-es2015 --save-dev
8. npm install gulp-plumber --save-dev

Run the "gulp" command in this folder to transpile. The task can be left running for live transpiling.
